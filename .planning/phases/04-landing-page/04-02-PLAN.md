   - `handleSubmit` function:
     a. Guard: if `!email.trim()` or `state === 'loading'`, return
     b. Set state to `'loading'`
     c. `fetch('/api/waitlist', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ email }) })`
     d. If `!res.ok`, throw
     e. Set state to `'done'`
     f. Catch: set state to `'error'`
   - Render when `state === 'done'`: green checkmark + "You're on the list!" text (emerald-400)
   - Render when `state === 'error'`: show input with red error text below saying "Something went wrong. Try again."
   - Default render: email input + "Join waitlist" button with ArrowRight icon
   - Input: `type="email"`, `onKeyDown` for Enter, dark-mode styling matching login page input (`bg-white/[0.04] border border-white/[0.10] focus:border-white/[0.22] rounded-lg px-4 py-2.5 text-sm text-white placeholder:text-white/25 outline-none`)
   - Button: white bg, dark text, rounded-lg, font-semibold, shows Loader2 spinner when loading
   - Do NOT use HTML `<form>` tag — use controlled `<input>` with `onClick`/`onKeyDown` handlers (CLAUDE.md mandate)
